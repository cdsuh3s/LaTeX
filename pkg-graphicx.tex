\begin{document}

\section{Including Graphics}

A picture can be imported using the \verb|graphicx| package
and the \verb|\includegraphics| command.

% Using center environment,
% an image can be put at the horizontal center on the page.
\begin{center}
  \includegraphics[height=5cm]{Ghostscript_Tiger-512px}
  % EPS, PNG, JPG, and PDF files can be braught in.
  % If the file extension is not given, the graphicx package will try to guess it.
  % If  more than a version of graphic exist, provide the extension you want.
\end{center}
